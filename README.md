# CAR-Rental-Management-System
Overview

SmartDrive Rentals is a Java desktop application developed using Java Swing and JDBC.
The system allows users to register rental cars and view stored records from a MySQL database.
This project demonstrates object-oriented programming, GUI development, and database integration using Java.

Features
Register new cars into the system
View all registered cars in a tabular format
Clear form inputs
Persistent data storage using MySQL
JDBC connectivity using prepared statements
Clean graphical user interface with Nimbus Look and Feel
Separation of concerns using the DAO pattern

Technologies Used:

Java (JDK 8 or higher)

Java Swing

JDBC (MySQL Connector/J)

MySQL Server 8

MySQL Workbench

Visual Studio Code

Application Description
Car Registration

The main form allows the user to input car details including register number, brand, model, year, and rental price.
Upon clicking the Register Car button, the data is validated and stored in the MySQL database.

View Cars

The View Cars feature displays all stored car records in a table using data retrieved from the database.
